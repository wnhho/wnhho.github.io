Oracle 常用语句

发表于 2019-06-22

运维小伙伴接触最多的就是linux和oracle了,学习总结下常用oracle语法如下

SQL的分类

1-数据操纵语言DML
主要有四种形式：

语句：

语法名称 含义
CREATE DATABASE 创建数据库
CREATE TABLE 创建表
DROP TABLE 删除表
ALTER TABLE 修改表的结构
CREATE VIEW 创建视图
DROP VIEW 删除视图
CREATE INDEX 创建索引
DROP INDEX 删除索引

2-数据控制语言DCL,数据控制语言用来授权或这回收访问数据库的权限
1) GRANT : 授权
2）REVOKE : 取消授权
3）ROLLBACK[WORK] TO [SAVEPOINT]:回退到某一点
回滚—ROLLBACK
4)COMMIT :提交
在数据库的插入和删除和修改时候，只有提交到数据库才算完成。
提交的类型有三种：显式提交，隐式提交，和自动提交。
显示提交：commit
隐式提交：alter,connect,audit,create,disconnect,drop,exit,grant,noaudit,quit,revoke,rename
自动提交：若把autocommit 设置为ON就是自动提交。

1.1基本的select 语句

1-对于查询的列字段数字是空的值情况时候，该列进行加减乘除，返回的是空值或者位置，对空值进行连接字符串的操作后，返回为被连接的字符串。

2-定义列别名

列别名有如下的作用：
1）改变列标题的名字
2）可用于计算结果
3）在列名和别名之间有选项AS关键字
4）如果别名种包含空格，或者特殊字符，或者大小写敏感，要求使用双引号。

select department_id did ,department_name dname from test; did与dname是对应的别名
select department_id as did ,department_name as dname from test; 可以使用as来表示
select department_id as “did” ,department_name as “dn ame” from test; 别名有空格使用双引号来表示。
select department_id 部门码，department_name as 部门名称 from test;

在一个SQL语句中，一个列别名，既能用在SELECT子句当中，也能用在ORDER BY 子句当中。不能在where当中使用。
3-连接运算符
连接运算符：
1）用两个竖线表示||
2）构造一个字符表达式的合成列

文字运算符：
文字字符串，是包含在select列表中的一个字符串，一个数字或者一个日期，并且不是列名或者
别名，日期和字符文字，必须放在单引号中（‘ ’），数字不需要。

3-相同的行：
如何去除相同的行呢，在select中distinct关键字，可以去除相同的行。

第一种去重，针对所有的列

第二种去重，针对某个列

4-条件和排序
使用WHERE子句
比较条件和between条件
IN,LIKE条件
NULL/NOT NULL 条件
逻辑条件与优先规则

WHERE子句能够比较列值，文字值，算数表达式，或者函数，WHERE子句由三个元素组成 : 列名，比较条件，常量或值列表。

在WHERE子句中字符串和日期必须包含在单引号 (‘ ’) 中。但是，数字常数不应该包含在单引号中。

ORACLE SQL中的比较条件包含以下几种运算符号：

语法名称 含义
= 等于
> 大于
>= 大于等于
< 小于
<= 小于等于
<> 不等于
！= 不等于
^= 不等于
between…..and 在两个值之间
IN(set) 匹配一个任意值列表
LIKE 匹配一个字符模板
is null 是一个空值
is not null 不是一个空值

如果IN条件中的成员是字符或日期，它们必须放在单引号 (‘’) 中
Select Employee_Id, Manager_Id, Department_Id From Employees Where Last_Name In (‘Hartstein’, ‘Vargas’);
